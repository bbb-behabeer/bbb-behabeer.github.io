# 名前について
和辛子びびびと申します。なぜこの名前になったのか。
あるVtuberが罰ゲームで腹筋ベルトを使う配信をしていました。
そのVTuberは腹筋ベルトを使うことを腹筋ビビビといっていました。
当時、リアルのあだ名由来のbbbというハンドルネームを使っており、
ちょうどよい読み方だと思ってビビビを名乗り始めました。
そのあとはやわらかい印象にしたくて、
ひらがなのびびびを使い、
VTuberになりたくて苗字を付けました。
なぜ苗字を和辛子にしたのかは…察してください。
